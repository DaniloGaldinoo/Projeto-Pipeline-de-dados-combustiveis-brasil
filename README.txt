Pipeline de dados de preços de combustíveis no Brasil!

Descrição:
Este projeto implementa um pipeline de engenharia de dados que coleta,
processa e visualiza dados de preços de combustíveis no Brasil.

O pipeline utiliza Python para extração e transformação de dados,
Apache NiFi para orquestração do fluxo de dados, MySQL como banco
de armazenamento e Apache Superset para criação de dashboards.
